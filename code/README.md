# Code

This directory is for implementation experiments.

Suggested projects:

## GMM

- Implement EM from scratch.
- Fit a mixture of Gaussians to 2D data.
- Visualize learned clusters.

## VAE

- Implement a simple VAE on MNIST.
- Visualize the latent space.
- Generate samples from random latent vectors.

## Diffusion

- Implement a toy DDPM on 2D points or MNIST.
- Visualize forward noising and reverse denoising.

## Transformers

- Implement a tiny character-level language model.
- Add causal masking.
- Add sampling with temperature, top-k, and top-p.

## SSM

- Implement a basic recurrent state update.
- Compare memory use against attention.

## Flows

- Implement a simple affine coupling layer.
- Train a normalizing flow on 2D data.
